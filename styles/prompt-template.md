# Image Generation Prompt Template

## Template

```
A cute 3D rendered {icon_subject} icon in clay morphism style, matching the style of corporate 3D app icons. The object must look like a smooth, simplified clay/plastic toy — NOT photorealistic. Surfaces should be clean and smooth with minimal texture detail, like a molded clay figurine. Use bold, saturated pastel colors rather than realistic muted colors. The object has smooth rounded edges with soft matte finish. Soft lighting from the upper-left creating gentle shading and highlights on the top-left surface for depth, but NO drop shadow on the ground. The object is slightly tilted at a 15-20 degree angle, viewed from a 3/4 perspective (slightly above). Plain white background (#FFFFFF). The object occupies about 50-70% of the square canvas with generous padding around it. Clean, minimal composition with no text, no watermarks. Single centered 3D object only — do NOT add any app icon frame, rounded rectangle container, or card-like background behind the object. The object must float freely on the white background with nothing surrounding it. Professional 3D illustration style — think Blender clay render, not food photography.
```

## 스타일 핵심 원칙

- **클레이 모핑**: 실물 그대로가 아닌, 클레이/플라스틱 장난감처럼 단순화된 3D
- **텍스처**: 최소화. 나무결·피부결 등은 암시 정도만, 사실적 질감은 금지
- **색상**: 선명하고 채도 높은 파스텔 톤 (실물 색상이 아닌 스타일화된 색상)
- **배경**: 순백 (#FFFFFF), 그라데이션 없음
- **그림자**: 물체 자체의 명암(셀프 섀도우)만 허용, 바닥 드롭 섀도우 없음
- **마감**: 매트 위주, 유기물(음식/식물/동물)에는 반짝임 없음

## 치환 규칙

- `{icon_subject}` → Phase 3 리서치 기반 상세 묘사로 치환
- 단순 단어가 아닌, 형태·색상·핵심 디테일을 포함한 구체적 설명
